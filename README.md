🏨 Hotel Booking Web Application

A web-based hotel booking system built using PHP and MySQL with XAMPP as the local development environment.
This project demonstrates full-stack integration with form handling, backend processing, and database storage.

🚀 Project Overview

This application allows users to submit hotel booking details through a web interface.
The data is processed using PHP and stored in a MySQL database for further management and retrieval.

🛠 Tech Stack
Frontend: HTML, CSS
Backend: PHP
Database: MySQL
Server: XAMPP (Apache, phpMyAdmin)
⚙️ Features
Clean and structured booking form
Backend form processing using PHP
MySQL database integration
Data storage and retrieval via phpMyAdmin
Local server deployment using XAMPP
🗄 Database
Database handled using MySQL
Tables managed via phpMyAdmin
Structured storage for booking records

Booking Form UI
Submitted Data Processing
phpMyAdmin Database View
XAMPP Server Running
▶️ How to Run Locally
Install XAMPP
Move project folder to htdocs
Start Apache and MySQL from XAMPP
Open browser and go to:
http://localhost/hotel-booking/
Import database file into phpMyAdmin (if included)

📌 Project Purpose

This project was developed to understand real-world full-stack web application flow including frontend design, backend logic, and database operations.
