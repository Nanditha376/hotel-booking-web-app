# 🏨 Hotel Booking App

A simple hotel booking system built with PHP and MySQL.

## Features
- View hotel rooms
- Book a room
- Save bookings to database

## Tech Stack
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL

## Files
- `index.php` – Main page
- `save.php` – Handles form submission
- `db.php` – Database connection

## Setup
1. Clone this repo
2. Import the SQL database
3. Update `db.php` with your DB credentials
4. Run on a local server (XAMPP / WAMP)
